# Pathfinder.

국민대학교 조형대 전과를 준비하는 학생들을 위한 모바일 우선 정보 플랫폼 프로토타입입니다. 공식 공지에는 잘 드러나지 않는 후기, 포트폴리오, 면접 자료, 자료 판매 신청 흐름을 한 곳에서 탐색할 수 있도록 설계했습니다.

## 주요 기능

- 관심 학과 기준 홈 콘텐츠 탐색
- 후기 목록, 후기 상세, 학과별 후기 필터
- 유료 자료 마켓, 자료 상세, 결제 모달 목업
- 학과 상세 페이지의 후기 기반 AI 인사이트
- 후기 작성 화면과 자료 판매 신청 안내 플로우
- 모바일 앱형 하단 탭 내비게이션

## 기술 스택

- React
- Vite
- Phosphor Icons
- CSS

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 Vite가 안내하는 로컬 주소로 접속하면 됩니다.

## 빌드

```bash
npm run build
```

## 참고

이 프로젝트는 발표와 시연을 위한 프론트엔드 프로토타입입니다.

- 모든 데이터는 더미데이터입니다.
- 실제 로그인, 서버 저장, 결제는 구현되어 있지 않습니다.
- 공식 전형 정보는 매 학기 변경될 수 있으므로 반드시 ON국민 및 국민대학교 학사공지에서 최신 공지를 확인해야 합니다.
